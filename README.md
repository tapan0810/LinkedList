## 📌 Problem Statement

Implement a **Singly Linked List** and perform the following operations:

1. Insert a node at the beginning (Head)  
2. Insert a node at the end (Tail)  
3. Insert a node at a given position  

---

## 🧠 Approach

- Create a `Node` structure with:
  - `data`
  - `next` pointer  
- Maintain a `head` pointer  
- Traverse the list to reach the required position  
- Update links carefully to avoid breaking the list  

### Edge Cases Covered:

- Empty list  
- Insertion at position 1  
- Insertion beyond list length
